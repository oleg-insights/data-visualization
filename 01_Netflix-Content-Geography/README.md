# Анализ географии производства контента

### Решаемая проблема:
Данные по странам-производителям контента 'замусорены', это не позволяет компании сфокусировать бюджет на локализации лидеров и приводит к лишним расходам.

### Решение:
1. Проведена очистка данных от лишних признаков (оставлена только основная страна-производитель).
2. Сформирован список топ-10 стран-производителей контента.
3. Построена диаграмма с показателями количества контента и акцентом на лидеров рынка.

### Инструменты:
Pandas, Matplotlib/Seaborn.

### Результат:
Можно руководствоваться списком и диаграммой топ-10 стран для точечного распределения бюджета на локализацию контента.
